# Bronnen

Deze map bevat de gebruikte bronnen voor de bachelorproef.  
De PDF-bestanden bevatten **highlights** om belangrijke passages aan te duiden, verdere notaties zijn te vinden in het
bronnen_overzicht markdown bestand.

De markeringen dienen als ondersteuning tijdens het literatuuronderzoek en bij het schrijven van de bachelorproef.
